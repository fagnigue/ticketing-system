
## Description
Ticketing system built with NestJS + Prisma (MongoDB for test purposes), following a Clean Architecture approach.
[Nest](https://docs.nestjs.com/) framework TypeScript documentation.

## Installation

```bash
$ npm install
```

## Initialize Prisma ORM Client

```bash
$ npm run db:sync:schema
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
